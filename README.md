# ssr-demo


```
git clone https://github.com/u4bi-store/ssr-demo.git

cd ssr-demo

yarn global add create-razzle-app

yarn install

yarn start
```

`yarn start`

`yarn build`

`yarn start:prod`

`yarn test`
